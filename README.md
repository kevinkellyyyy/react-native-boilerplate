# BoilerplateRN

## Getting started

- npm install / yarn install
- npx react-native start
- npx react-native run-android

```

#Documentation

- Icon - [react-native-fontawesome](https://www.npmjs.com/package/@fortawesome/react-native-fontawesome)

- Local storage - [AsyncStorage](https://react-native-async-storage.github.io/async-storage/docs/install/) and [React native encrypted storage](https://github.com/emeraldsanto/react-native-encrypted-storage) (manual linking in IOS)

- Internet information - [NetInfo](https://github.com/react-native-netinfo/react-native-netinfo) (manual linking in IOS)

- HTTP client - [apisauce](https://github.com/infinitered/apisauce)

- Form - [Formik](https://formik.org/)

- Animation - [Lottie](https://github.com/lottie-react-native/lottie-react-native) (manual linking in IOS)

- Location - [Geolocation](https://github.com/Agontuk/react-native-geolocation-service)

- Permission - [react native permission](https://github.com/zoontek/react-native-permissions) (manual linking in IOS)

- Skeleton - [skeleton placeholder](https://www.npmjs.com/package/react-native-skeleton-placeholder)

- Validation - [Yup](https://www.npmjs.com/package/yup)
```
